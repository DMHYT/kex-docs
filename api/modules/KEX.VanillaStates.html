Namespace VanillaStates

Namespace containing Java wrappers for all the vanilla block states (properties).

Some of them are of Java enum type, should be converted to integer type using BlockStates.EnumProperty.toIntegerProperty method. There are JS enums in the BlockStates namespace provided for the most of them.
